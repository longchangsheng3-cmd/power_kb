# 术语表

| 术语 | 说明 |
|---|---|
| 待机功耗 | 设备在锁屏或低活动状态下的功耗 |
| Deep Sleep | 平台进入深度低功耗状态 |
| Suspend | 系统挂起流程或状态 |
| Wakelock | 阻止系统进入睡眠或唤醒系统的锁/源 |
| Wakeup Source | 能够唤醒系统的事件源 |
| Suspend Blocker | 阻止系统完成 suspend 的因素 |
| Alarm | 定时唤醒机制，可能来自系统或应用 |
| PMIC | 电源管理芯片 |
| Regulator | 电源轨或稳压器控制对象 |
| GPIO Leakage | GPIO 电平或上下拉配置导致的漏电 |
