## 问题现象

锁屏待机后平均电流高于基准，电流曲线存在周期性尖峰，系统深睡占比偏低。

## 输入材料

- wakeup_sources 显示 wlan 相关 source 触发次数持续增加。
- kernel log 中存在周期性网络唤醒痕迹。
- 飞行模式下功耗恢复正常。

## 分析过程

1. 对比飞行模式和 Wi-Fi 连接场景，确认异常与网络相关。
2. 查看 wakeup_sources，发现 wlan source 高频触发。
3. 检查 AP 环境和应用心跳，确认是否存在网络包唤醒。

## 根因

示例根因：Wi-Fi 场景下存在周期性网络包唤醒，导致系统频繁退出低功耗状态。

## 优化方案

- 检查 Wi-Fi power save 配置。
- 分析网络包来源，减少不必要心跳。
- 对后台应用或服务增加待机策略限制。
