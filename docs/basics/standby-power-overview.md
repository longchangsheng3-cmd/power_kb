# 待机功耗概览

## 定义

待机功耗是设备在屏幕关闭、用户无交互、业务进入低活动状态后仍然消耗的电流或功率。它通常受系统睡眠状态、唤醒源、外设状态、网络行为和应用后台活动影响。

## 常见指标

- 待机平均电流：一段时间内的平均电流。
- 深睡占比：系统处于 deep sleep 或 suspend 状态的时间比例。
- 唤醒频率：单位时间内被中断、alarm、网络或外设唤醒的次数。
- 峰值电流：周期性唤醒或外设活动造成的瞬时电流。

## 常见异常形态

| 现象 | 可能方向 |
|---|---|
| 平均电流持续偏高 | 无法进入深睡、外设未下电、后台任务活跃 |
| 周期性电流尖峰 | alarm、网络心跳、传感器轮询、日志刷新 |
| 深睡占比低 | wakelock、suspend blocker、频繁中断 |
| 单板差异大 | 硬件漏电、外设状态、校准或器件批次差异 |

## 基础分析思路

1. 明确测试场景和基准值。
2. 对比正常样机和异常样机。
3. 先判断系统是否进入目标睡眠状态。
4. 再分析唤醒源、外设、网络和应用行为。
5. 修改后用相同测试条件复测。
