# 待机功耗测试方法

## 测试前准备

- 确认软件版本、硬件版本和配置一致。
- 记录电池电量、温度、网络环境和外设连接状态。
- 关闭不必要的调试功能，或明确记录调试开关状态。
- 使用稳定电源、功耗仪或电流采集设备。

## 推荐测试流程

1. 设备开机并完成初始化。
2. 设置目标场景，例如飞行模式、Wi-Fi 连接、插 SIM、静置锁屏。
3. 等待系统稳定后开始采集。
4. 采集足够长时间，覆盖周期性唤醒。
5. 导出电流曲线和系统日志。

## 常用数据

- 电流曲线。
- kernel log。
- wakeup_sources。
- suspend_stats。
- alarm dump。
- 电源管理服务日志。
- Wi-Fi、BT、GNSS、Modem 相关日志。

## 注意事项

- 不同网络环境会显著影响待机功耗。
- USB 调试和日志开关可能改变系统睡眠行为。
- 对比测试必须保证场景一致。
- 单次测试不足以证明优化有效，建议多轮复测。
