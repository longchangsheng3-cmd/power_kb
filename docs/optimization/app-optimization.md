# 应用与服务待机优化

## 优化方向

- 减少后台定时任务和网络心跳。
- 限制不必要的定位、传感器和扫描请求。
- 合并周期性任务，避免频繁唤醒。
- 遵循系统后台管理和省电策略。

## 检查重点

| 方向 | 检查项 |
|---|---|
| Alarm | 是否周期过短，是否可延后或合并 |
| Network | 是否存在高频心跳或轮询 |
| Location | 是否后台持续定位 |
| Sensor | 是否锁屏后仍订阅传感器 |
| Log | 是否持续写日志或刷新状态 |

## 验证方法

- 对比卸载或禁用应用前后的功耗。
- 检查 alarm dump 和后台任务统计。
- 检查网络包和唤醒源变化。
- 做长时间锁屏复测。
