# Modem 与网络待机优化

## 优化方向

- 确认 Modem 低功耗状态是否进入。
- 优化网络注册、寻呼、心跳和重选行为。
- 检查弱信号环境下的功耗变化。
- 检查 Wi-Fi、BT、GNSS 的低功耗配置。

## 检查重点

| 场景 | 重点 |
|---|---|
| 插 SIM 待机高 | 信号强度、网络制式、Modem sleep 状态 |
| Wi-Fi 待机高 | power save、AP 行为、网络包唤醒 |
| BT 待机高 | 连接状态、扫描、广播间隔 |
| GNSS 待机高 | 定位请求、辅助定位、后台服务 |

## 验证方法

- 对比飞行模式、插 SIM、Wi-Fi、BT、GNSS 场景。
- 更换网络环境和 AP。
- 分析唤醒源和模块日志。
- 修改配置后复测平均电流和尖峰频率。
