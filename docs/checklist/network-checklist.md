# 网络待机检查清单

## 适用场景

插 SIM、连接 Wi-Fi、开启蓝牙或 GNSS 后待机电流偏高，或电流曲线出现周期性尖峰。

## 需要收集的信息

- 网络制式、信号强度和小区环境。
- Wi-Fi AP 型号、频段和省电模式。
- BT/GNSS 使用状态。
- Modem、Wi-Fi、BT 相关日志。
- 电流曲线和唤醒源统计。

## 排查步骤

1. 对比飞行模式、Wi-Fi、蜂窝网络等不同场景。
2. 检查是否存在网络包唤醒或心跳过密。
3. 检查 Wi-Fi power save 是否生效。
4. 检查 Modem 是否进入低功耗状态。
5. 检查 BT/GNSS 是否有后台扫描或定位请求。
6. 更换网络环境做复测。

## 常见原因

| 现象 | 可能原因 | 处理建议 |
|---|---|---|
| Wi-Fi 场景周期性尖峰 | AP 或应用心跳唤醒 | 分析网络包和 wakelock |
| 插 SIM 后平均电流高 | 信号差或 Modem 未低功耗 | 更换环境并检查 Modem 日志 |
| GNSS 相关唤醒 | 后台定位未释放 | 检查定位请求来源 |
